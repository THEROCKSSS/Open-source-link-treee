# 🌐 Futuristic Link Tree - Multi-Game Template Platform

A modern, futuristic link tree application with support for multiple templates optimized for different gaming platforms and communities. Create stunning link aggregation pages that work in VR environments and beyond.

---

## 🎯 Project Overview

This project provides a customizable, web-based link tree solution with multiple futuristic template designs. Each template is optimized for specific use cases while maintaining a cohesive futuristic aesthetic.

**Key Features:**
- 🎮 Multi-game template support (VR Chat, Discord, Twitch, etc.)
- ✨ Smooth animations and interactive elements
- 🌙 Dark-mode focused aesthetic
- 📱 Responsive design (desktop, mobile, VR headset displays)
- ⚙️ Easy JSON configuration system
- 🚀 GitHub Pages compatible (static deployment)
- 🎨 Fully customizable colors and styling

## ⚙️ Configuration System

Each template uses a simple JSON configuration file:

```json
{
  "profile": {
    "name": "Your Username",
    "bio": "Your bio text",
    "avatar": "path/to/avatar.png"
  },
  "links": [
    {
      "title": "Link Title",
      "url": "https://example.com",
      "icon": "icon-name",
      "category": "social"
    }
  ],
  "colors": {
    "primary": "#00F0FF",
    "secondary": "#9D00FF",
    "background": "#0A0E27"
  },
  "animations": true,
  "template": "vrchat-dimension"
}
```

## 🌐 GitHub Deployment

This project is designed to be fully GitHub Pages compatible:

1. **Push to GitHub:** `git push origin main`
2. **Enable GitHub Pages:** Go to repo settings → Pages → select `main` branch
3. **Access:** Your site will be available at `https://yourusername.github.io/repo-name`
4. **Custom Domain:** (Optional) Point custom domain to GitHub Pages

**No build process required** - pure HTML/CSS/JS
