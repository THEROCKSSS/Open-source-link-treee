## 🔧 Configuration System Explained

### Global Config (config.json)
```json
{
  "version": "1.0.0",
  "defaultTemplate": "basic-clean",
  "templateOptions": [6 template names],
  "features": { enabled features }
}
```

### Template Config (config-example.json)
```json
{
  "profileName": "Your Name",
  "bio": "Your bio",
  "links": [{ platform, title, url }, ...],
  "customization": { colors, animations, etc }
}
```

### Platform Reference (icons-manifest.json)
```json
{
  "platforms": [
    {
      "id": "twitter",
      "name": "Twitter/X",
      "color": "#1DA1F2",
      "icon": "twitter.svg"
    }
  ]
}
```
